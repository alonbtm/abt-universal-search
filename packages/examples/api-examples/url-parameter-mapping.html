URL Parameter Mapping Examples

This guide demonstrates different strategies for mapping search queries to URL parameters in REST API calls.

Parameter Mapping Strategies

1. Simple Query Parameter
Best for: Basic search functionality, simple APIs
// Configuration { queryParam: 'q', requestTransform: (query) => ({ url: `${endpoint}?q=${encodeURIComponent(query)}`, method: 'GET' }) }
GET /api/search?q=javascript
2. Named Field Search
Best for: Field-specific search, filtering by attribute
// Configuration { queryParam: 'name_like', requestTransform: (query) => ({ url: `${endpoint}?name_like=${encodeURIComponent(query)}`, method: 'GET' }) }
GET /api/users?name_like=john
3. Multiple Parameter Search
Best for: Complex queries, multiple search fields
// Configuration { requestTransform: (query) => { const params = new URLSearchParams({ search: query, fields: 'name,description,tags', limit: '10', sort: 'relevance' }); return { url: `${endpoint}?${params.toString()}`, method: 'GET' }; } }
GET /api/search?search=javascript&fields=name%2Cdescription%2Ctags&limit=10&sort=relevance
4. Filter-Based Search
Best for: Advanced filtering, structured queries
// Configuration { requestTransform: (query) => { const filters = { 'filter[name]': query, 'filter[status]': 'active', 'include': 'category,tags' }; const params = new URLSearchParams(filters); return { url: `${endpoint}?${params.toString()}`, method: 'GET' }; } }
GET /api/items?filter%5Bname%5D=javascript&filter%5Bstatus%5D=active&include=category%2Ctags
5. REST Path Parameters
Best for: Hierarchical data, category-based search
// Configuration { requestTransform: (query) => { const encodedQuery = encodeURIComponent(query); return { url: `${endpoint}/search/${encodedQuery}`, method: 'GET' }; } }
GET /api/products/search/javascript
6. JSON:API Specification
Best for: JSON:API compliant services
// Configuration { requestTransform: (query) => { const params = new URLSearchParams({ 'filter[search]': query, 'page[size]': '10', 'include': 'category', 'sort': '-updated_at' }); return { url: `${endpoint}?${params.toString()}`, method: 'GET', headers: { 'Accept': 'application/vnd.api+json' } }; } }
GET /api/articles?filter%5Bsearch%5D=javascript&page%5Bsize%5D=10&include=category&sort=-updated_at

Advanced Parameter Encoding

// Handling special characters and encoding function safeEncodeQuery(query) { // Basic URL encoding const basicEncoded = encodeURIComponent(query); // Handle special cases for different APIs const strategies = { // Strategy 1: Basic encoding (most common) basic: basicEncoded, // Strategy 2: Plus encoding for spaces (some legacy APIs) plusSpaces: query.replace(/ /g, '+').replace(/[^\w+]/g, encodeURIComponent), // Strategy 3: Custom encoding for special chars customEncoding: query .replace(/&/g, '%26') .replace(/=/g, '%3D') .replace(/#/g, '%23') .replace(/\?/g, '%3F'), // Strategy 4: Base64 encoding for complex queries base64: btoa(query), // Strategy 5: JSON string encoding jsonString: JSON.stringify({ query: query }) }; return strategies; } // Example usage in requestTransform requestTransform: (query) => { const encoded = safeEncodeQuery(query); return { url: `${endpoint}?q=${encoded.basic}`, method: 'GET' }; }

Error Handling for Parameter Mapping

// Robust parameter mapping with error handling function createParameterMapper(config) { return (query) => { try { // Validate query if (!query || typeof query !== 'string') { throw new Error('Invalid query parameter'); } // Apply length limits if (query.length > config.maxQueryLength) { query = query.substring(0, config.maxQueryLength); } // Sanitize query const sanitizedQuery = query.trim().replace(/[<>]/g, ''); // Build URL based on strategy switch (config.strategy) { case 'simple': return buildSimpleUrl(sanitizedQuery, config); case 'complex': return buildComplexUrl(sanitizedQuery, config); default: throw new Error(`Unknown strategy: ${config.strategy}`); } } catch (error) { console.error('Parameter mapping error:', error); // Return fallback URL return { url: `${config.endpoint}?q=${encodeURIComponent(query || '')}`, method: 'GET' }; } }; }
