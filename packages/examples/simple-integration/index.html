Universal Search - Hello World Example

This demonstrates the basic functionality of the Universal Search component with in-memory data.

Selected:

Implementation (Under 5 Lines!)

const userData = [/* user objects */];
const search = new UniversalSearch('#user-search', {
  dataSource: { type: 'memory', data: userData, searchFields: ['name', 'email'] }
});
search.init();

Search Features

  • Type to search users by name, email, or role
  • Navigate results with arrow keys
  • Press Enter to select
  • Click to select a result
  • Minimum 2 characters required
  • 300ms debounce for performance
